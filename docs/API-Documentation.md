## Private JS APIs

These APIs are currently undocumented and reserved for internal use. They are usually available to components in the system hence don't require the user to access them directly.

- Fliplet Runtime
- Fliplet Interact
- Fliplet Menu
- Fliplet Native
- Fliplet Security
- Fliplet App Submissions
- Fliplet Chat
